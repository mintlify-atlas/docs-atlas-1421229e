  <div className="mb-12">
    <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
    <p className="text-base text-gray-600 dark:text-gray-400">Get up and running with your first reliable agent in minutes</p>
  </div>

  <Steps>
    <Step title="Install Dependencies">
      Choose your preferred language and install the required packages.
      
      <CodeGroup>
      ```bash Python (uv)
      git clone --depth 1 https://github.com/langchain-ai/lca-reliable-agents.git
      cd lca-reliable-agents/python
      uv sync
      ```
      
      ```bash TypeScript
      git clone --depth 1 https://github.com/langchain-ai/lca-reliable-agents.git
      cd lca-reliable-agents/ts
      npm install
      ```
      </CodeGroup>
    </Step>

    <Step title="Configure API Keys">
      Set up your environment variables with your OpenAI and LangSmith API keys.
      
      ```bash .env
      OPENAI_API_KEY='your_openai_api_key_here'
      LANGSMITH_API_KEY='your_langsmith_api_key_here'
      LANGSMITH_TRACING=true
      LANGSMITH_PROJECT=lca-reliable-agents
      ```
      
      <Note>
      Get your [OpenAI API key](https://openai.com/index/openai-api/) and create a free [LangSmith account](https://smith.langchain.com/) to get your LangSmith API key.
      </Note>
    </Step>

    <Step title="Run Your First Agent">
      Start the OfficeFlow customer support agent with full tracing enabled.
      
      <CodeGroup>
      ```bash Python
      uv run python officeflow-agent/agent_v5.py
      ```
      
      ```bash TypeScript
      npx tsx officeflow-agent/agent_v5.ts
      ```
      </CodeGroup>
      
      The agent will start an interactive session where you can ask questions about products, inventory, and company policies.
    </Step>

    <Step title="View Traces in LangSmith">
      Open [LangSmith](https://smith.langchain.com/) and navigate to your project to see detailed traces of every agent interaction, including LLM calls, tool usage, and decision paths.
    </Step>
  </Steps>
</div>
